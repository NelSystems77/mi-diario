🔧 PROBLEMA COMÚN Y SOLUCIÓN:

❌ PROBLEMA: La colección "users" no aparece en Firebase

✅ SOLUCIÓN:
   1. Abre Firebase Console
   2. Ve a Firestore Database → Reglas
   3. Copia el contenido de "firestore-dev.rules"
   4. Pega y publica en Firebase Console
   5. Abre init-firebase.html en el navegador
   6. Sigue las instrucciones para crear usuario admin

   📖 Detalles completos en: SOLUCION_USERS.md
